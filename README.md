# AI Brand Voice Tweet Generator

This project generates tweets that match a brand's voice using AI.

## Features
- Brand voice analysis
- Tone identification
- Target audience detection
- Content theme generation
- AI-generated tweets for social media campaigns

## Tech Stack
- Python
- Streamlit
- Groq API
- LLaMA 3.1 Model

## How It Works
1. User enters brand details
2. AI analyzes brand voice
3. AI generates tweets aligned with tone and audience

## Run Locally

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run app.py
